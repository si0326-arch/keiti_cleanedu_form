# KEITI 신고서 작성 체험

한국환경산업기술원 청렴 교육용 신고서 작성 시뮬레이션입니다.

## 소개
청탁금지법·이해충돌방지법·갑질방지 등 KEITI 임직원이 알아야 할
청렴 신고서를 시나리오와 함께 직접 작성해보는 학습 체험 도구입니다.

## 주요 기능
- 소속 처·단별 1순위 부패위험 맞춤 신고서 추천
- 실제 사례 기반 시나리오 2종 제공
- 작성 완료 시 구글 시트 자동 저장
- 학습 효과 측정 설문 포함

## 사용 방법
아래 링크에 접속하여 체험하세요.

- 권익위 사례집 / 이충법 유권해석집 / 청탁금지법 결정례 기반
- 본 페이지는 학습용 시뮬레이션입니다
- 실제 신고는 본원 청탁방지담당관에게 정식 제출하시기 바랍니다
